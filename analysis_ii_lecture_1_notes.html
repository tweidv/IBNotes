Setup: Sequences of Functions

Let $E$ be an arbitrary set (for the moment). Consider a sequence of functions $f_n : E \to \mathbb{R}$ for $n = 1, 2, 3, \ldots$, where each function is defined on $E$ and takes real values.

This could also be done with complex-valued functions, but we will focus on real-valued functions for now.

▶10:45

Pointwise Limit and Pointwise Convergence

Suppose that for any $x \in E$, the sequence of real numbers $f_n(x)$ converges to some real number. We can define a new function $f : E \to \mathbb{R}$ by:

$$f(x) = \lim_{n \to \infty} f_n(x)$$

This function $f$ is called the pointwise limit of the sequence $(f_n)$. We say that $(f_n)$ converges to $f$ pointwise on $E$.

The key idea is that we fix an arbitrary point $x \in E$ and require the sequence of real numbers $f_n(x)$ to converge. Since this holds for every $x$, the limit defines a function.

▶15:40

Preservation of Properties Under Pointwise Convergence

A natural question arises: if each $f_n$ has a useful property (such as continuity, differentiability, or integrability), does the pointwise limit $f$ also have that property?

For example: If $f_n$ is continuous on $E$ for every $n$, does it follow that $f$ is continuous on $E$?

Unfortunately, with pointwise convergence, this is not true. Continuity is not preserved, and neither are differentiability or integrability. Pointwise convergence is too weak a notion.

▶18:36

Continuity Not Preserved Under Pointwise Convergence

Consider the sequence of functions $f_n : [-1, 1] \to \mathbb{R}$ defined by:

$$f_n(x) = x^{2n+1}$$

Each $f_n$ is continuous on $[-1, 1]$ (being a polynomial). However, the pointwise limit is:

$$f(x) = \begin{cases} 1 & \text{if } x > 0 \\ 0 & \text{if } x = 0 \\ -1 & \text{if } x < 0 \end{cases}$$

This limit function $f$ is discontinuous at $x = 0$, with a jump discontinuity. We can visualize this as the functions getting steeper and steeper, eventually "collapsing" to the step function.

As $n$ increases, $x^{2n+1}$ approaches 0 for $|x| < 1$, approaches 1 for $x = 1$, and approaches -1 for $x = -1$. For $x$ near 0, the functions become very flat near 0 but then rise/fall steeply to approach the limiting step function.

▶22:11

Integrability Not Preserved Under Pointwise Convergence

Consider the interval $[0, 1]$ with the rationals enumerated as $q_1, q_2, q_3, \ldots$ (the rationals are countable). Define:

$$f_n(x) = \begin{cases} 1 & \text{if } x \in \{q_1, q_2, \ldots, q_n\} \\ 0 & \text{otherwise} \end{cases}$$

Each $f_n$ is Riemann integrable on $[0, 1]$ because it is continuous everywhere except at a finite set of points (namely, the first $n$ rationals). A result from Analysis I states that functions continuous except on a finite set are integrable.

However, $(f_n)$ converges pointwise to:

$$f(x) = \begin{cases} 1 & \text{if } x \in \mathbb{Q} \\ 0 & \text{otherwise} \end{cases}$$

This limit function is the characteristic function of the rationals, which is not Riemann integrable on $[0, 1]$ (another result from Analysis I).

▶28:14

Integrals Do Not Converge Under Pointwise Convergence

Even if $f_n$ converges pointwise to $f$ and both $f_n$ and $f$ are integrable, the integrals may not converge. That is, it may not be true that:

$$\int_0^1 f_n(x) \, dx \to \int_0^1 f(x) \, dx$$

Consider a sequence of "triangle" functions on $[0, 1]$:

$$f_n(x) = \begin{cases} n^2 x & \text{if } 0 \le x \le \frac{1}{n} \\ n^2 \left(\frac{2}{n} - x\right) & \text{if } \frac{1}{n} < x \le \frac{2}{n} \\ 0 & \text{otherwise} \end{cases}$$

This is an isosceles triangle with base $[0, \frac{2}{n}]$ and height $n$. The area (integral) of each triangle is:

$$\int_0^1 f_n(x) \, dx = 1$$

However, for any fixed $x > 0$, eventually the triangle will be entirely to the left of $x$ (since the base width $\frac{2}{n} \to 0$), so $f_n(x) \to 0$. At $x = 0$, $f_n(0) = 0$ for all $n$. Therefore:

$$f(x) = \lim_{n \to \infty} f_n(x) = 0 \text{ for all } x \in [0, 1]$$

This gives $\int_0^1 f(x) \, dx = 0$, but $\int_0^1 f_n(x) \, dx = 1$ for all $n$. The integrals do not converge!

▶34:37

Need for a Stronger Notion of Convergence

Pointwise convergence is not strong enough to ensure that useful properties like continuity and integrability are preserved in the limit. We need a stronger notion of convergence for sequences of functions.

However, the notion cannot be too strong. For example, if we required that $f_n$ is eventually identically equal to $f$, then all properties would be preserved—but this is too restrictive to be interesting or useful.

The key challenge in mathematics is finding the "happy medium": a class of objects or a notion that is:

  • Rich enough to include many interesting examples
  • Restrictive enough to allow useful theorems to be proved

For convergence of function sequences, the happy medium is uniform convergence.

Uniform Convergence

Let $(f_n)$ be a sequence of real-valued functions defined on a set $E$, and let $f : E \to \mathbb{R}$ be another function. We say that $(f_n)$ converges to $f$ uniformly on $E$ if:

For every $\epsilon > 0$, there exists $N$ (depending on $\epsilon$) such that for every $n \ge N$ and for every $x \in E$, we have:

$$|f_n(x) - f(x)| < \epsilon$$

The crucial point is that the same $N$ works for all $x \in E$ simultaneously. The value $N$ depends only on $\epsilon$, not on the choice of $x$.

▶43:01

Equivalent Formulation of Uniform Convergence

An extremely useful equivalent way to express uniform convergence is:

$(f_n)$ converges to $f$ uniformly on $E$ if and only if:

$$\sup_{x \in E} |f_n(x) - f(x)| \to 0 \text{ as } n \to \infty$$

This formulation is very practical: to check uniform convergence, compute the supremum of $|f_n - f|$ over all $x \in E$. This gives a sequence of real numbers (depending on $n$). If this sequence tends to 0, then $(f_n)$ converges to $f$ uniformly.

Comparison: Uniform vs. Pointwise Convergence

Let's compare the two notions in terms of their $\epsilon$-$N$ definitions.

Pointwise convergence: $f_n(x) \to f(x)$ for each $x \in E$ means:

For every $x \in E$ and every $\epsilon > 0$, there exists $N$ (depending on both $\epsilon$ and $x$) such that for all $n \ge N$:

$$|f_n(x) - f(x)| < \epsilon$$

Here, $N$ is allowed to depend on both $\epsilon$ and $x$. Different points may require different values of $N$.

Uniform convergence: For every $\epsilon > 0$, there exists $N$ (depending only on $\epsilon$) such that for all $n \ge N$ and for all $x \in E$ simultaneously:

$$|f_n(x) - f(x)| < \epsilon$$

The key difference: for uniform convergence, the same $N$ works for all $x$ given $\epsilon$. For pointwise convergence, $N$ can depend on $x$.

This seemingly small difference makes all the difference! In the next lecture, we will prove that uniform convergence is strong enough to preserve continuity and the values of integrals.
