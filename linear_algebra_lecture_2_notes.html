Complex Conjugation Question

From the previous lecture, we had a question about whether complex conjugation is a linear map. The answer depends on which field we're working over:

  • If we view $\mathbb{C}$ as a complex vector space: No, because complex conjugation doesn't respect scalar multiplication by complex numbers
  • If we view $\mathbb{C}$ as a real vector space: Yes, because it does respect scalar multiplication by real numbers

Isomorphism

An isomorphism $\alpha: V \to W$ between vector spaces $V$ and $W$ over a field $F$ is a bijective linear map.

We say that $V$ and $W$ are isomorphic if there exists an isomorphism from $V$ to $W$, denoted $V \cong W$.

Intuitively, if two vector spaces are isomorphic, then from the point of view of vector space structure, they are the same vector space - just with elements labeled differently.

▶6:45

Isomorphism Example

Let $V$ be the set of 4-dimensional column vectors over field $F$ and $W$ be the set of $2 \times 2$ matrices over $F$. Define the map $\alpha: V \to W$ by:

$$\alpha\begin{pmatrix} a \\ b \\ c \\ d \end{pmatrix} = \begin{pmatrix} a & b \\ c & d \end{pmatrix}$$

This is an isomorphism of vector spaces, showing that as far as vector space structure is concerned, $2 \times 2$ matrices are just an array of four numbers arranged differently.

Inverse of Isomorphism

If $\alpha: V \to W$ is an isomorphism, then $\alpha^{-1}: W \to V$ is also an isomorphism.

Proof: The inverse of a bijection is always a bijection. For linearity, let $w_1, w_2 \in W$ and $\lambda \in F$. Since $\alpha$ is surjective, we can write $w_i = \alpha(v_i)$ for some $v_i \in V$. Then:

$$\alpha^{-1}(w_1 + \lambda w_2) = \alpha^{-1}(\alpha(v_1) + \lambda \alpha(v_2)) = \alpha^{-1}(\alpha(v_1 + \lambda v_2)) = v_1 + \lambda v_2 = \alpha^{-1}(w_1) + \lambda \alpha^{-1}(w_2)$$

▶12:50

Isomorphism Properties

The relation of being isomorphic is an equivalence relation on vector spaces:

  • Reflexive: Every vector space is isomorphic to itself (via the identity map)
  • Symmetric: If $V \cong W$, then $W \cong V$ (via the inverse isomorphism)
  • Transitive: If $V \cong W$ and $W \cong U$, then $V \cong U$ (via composition of isomorphisms)

Kernel and Image

For a linear map $\alpha: V \to W$ between vector spaces $V$ and $W$ over field $F$:

  • Kernel of $\alpha$: $\ker(\alpha) = \{v \in V : \alpha(v) = 0_W\}$ (subset of $V$)
  • Image of $\alpha$: $\text{im}(\alpha) = \{\alpha(v) : v \in V\}$ (subset of $W$)

These definitions are analogous to what we saw in group theory, where the kernel consists of elements "killed off" by the map, and the image is what we get out at the other side.

▶16:18

Kernel and Image Properties

For any linear map $\alpha: V \to W$:

  1. $\ker(\alpha)$ is always a subspace of $V$
  2. $\text{im}(\alpha)$ is always a subspace of $W$
  3. $\alpha$ is surjective if and only if $\text{im}(\alpha) = W$
  4. $\alpha$ is injective if and only if $\ker(\alpha) = \{0_V\}$
▶18:05

Zero Vector Property

Claim: Any linear map takes the zero vector to the zero vector.

Proof: For any linear map $\alpha: V \to W$, we have:

$$\alpha(0_V) = \alpha(0_V + 0_V) = \alpha(0_V) + \alpha(0_V)$$

Subtracting $\alpha(0_V)$ from both sides gives $\alpha(0_V) = 0_W$.

This tells us that $0_V \in \ker(\alpha)$ and $0_W \in \text{im}(\alpha)$.

▶19:38

Injectivity Criterion

Claim: $\alpha$ is injective if and only if $\ker(\alpha) = \{0_V\}$.

Proof:

($\Rightarrow$): Suppose $\alpha$ is injective. We know $0_V \in \ker(\alpha)$. If $v \in \ker(\alpha)$, then $\alpha(v) = 0_W = \alpha(0_V)$. Since $\alpha$ is injective, $v = 0_V$. Therefore $\ker(\alpha) = \{0_V\}$.

($\Leftarrow$): Suppose $\ker(\alpha) = \{0_V\}$. Let $u, v \in V$ with $\alpha(u) = \alpha(v)$. Then $\alpha(u - v) = \alpha(u) - \alpha(v) = 0_W$, so $u - v \in \ker(\alpha) = \{0_V\}$. Therefore $u - v = 0_V$, so $u = v$. Hence $\alpha$ is injective.

Quotient Space

For a vector space $V$ over field $F$ and a subspace $W$ of $V$, the quotient space $V/W$ is the set of additive cosets of $W$ in $V$:

$$V/W = \{v + W : v \in V\}$$

Since $V$ is an abelian group under addition and $W$ is a normal subgroup, we can define vector space operations on $V/W$:

  • Addition: $(u + W) + (v + W) = (u + v) + W$
  • Scalar multiplication: $\lambda(v + W) = (\lambda v) + W$
▶25:21

Coset Equality Criterion

For vectors $u, v \in V$, the cosets $u + W$ and $v + W$ are equal if and only if $u - v \in W$.

This criterion is used to check when two cosets are the same, even when represented by different elements.

▶28:23

Well-Defined Operations

It's crucial to verify that the vector space operations on $V/W$ are well-defined. This means that if we represent the same cosets with different elements, the operations should give the same result.

For example, if $u + W = u' + W$ and $v + W = v' + W$, then $(u + v) + W = (u' + v') + W$ and $\lambda(u + W) = \lambda(u' + W)$.

▶32:51

Quotient Map

The quotient map $\pi_W: V \to V/W$ is defined by $\pi_W(v) = v + W$.

This map is:

  • Surjective: Every coset is the image of some vector
  • Linear: $\pi_W(u + \lambda v) = \pi_W(u) + \lambda \pi_W(v)$
  • Kernel: $\ker(\pi_W) = W$

The quotient map represents the "blurring out" process where information contained in $W$ is made unreadable.

First Isomorphism Theorem

For vector spaces $V$ and $W$ over field $F$ and a linear map $\alpha: V \to W$, there is an isomorphism:

$$\bar{\alpha}: V/\ker(\alpha) \to \text{im}(\alpha)$$

defined by $\bar{\alpha}(v + \ker(\alpha)) = \alpha(v)$.

This theorem says that what survives transmission through $\alpha$ is the same as what you would get if you took $V$ and blurred out the information in $\ker(\alpha)$.

▶40:25

First Isomorphism Theorem Proof

We need to check that $\bar{\alpha}$ is well-defined, injective, surjective, and linear.

Well-defined and injective: For $u, v \in V$, the cosets $u + \ker(\alpha)$ and $v + \ker(\alpha)$ are equal if and only if $u - v \in \ker(\alpha)$, which happens if and only if $\alpha(u - v) = 0$, which happens if and only if $\alpha(u) = \alpha(v)$, which happens if and only if $\bar{\alpha}(u + \ker(\alpha)) = \bar{\alpha}(v + \ker(\alpha))$.

Surjective: Given $w \in \text{im}(\alpha)$, there exists $v \in V$ such that $w = \alpha(v) = \bar{\alpha}(v + \ker(\alpha))$.

Linear: $\bar{\alpha}((u + \ker(\alpha)) + \lambda(v + \ker(\alpha))) = \bar{\alpha}(u + \lambda v + \ker(\alpha)) = \alpha(u + \lambda v) = \alpha(u) + \lambda \alpha(v) = \bar{\alpha}(u + \ker(\alpha)) + \lambda \bar{\alpha}(v + \ker(\alpha))$.

▶46:13

Span and Linear Combinations

For a vector space $V$ over field $F$ and a subset $S \subseteq V$:

  • Linear combination of $S$: An expression of the form $\sum_{s \in S} \lambda_s s$ where $\lambda_s \in F$ and only finitely many $\lambda_s$ are non-zero
  • Span of $S$: $\langle S \rangle = \{\text{all linear combinations of } S\}$
  • Spanning set: $S$ spans $V$ (or is a spanning set for $V$) if $\langle S \rangle = V$
  • Finite dimensional: $V$ is finite dimensional if it has a finite spanning set
▶47:19

Finite Sums Restriction

In linear algebra, we only allow finite sums of vectors. If $S$ is infinite, we require that all but finitely many coefficients $\lambda_s$ are zero, ensuring we're only adding finitely many terms.

This restriction is necessary because we haven't defined a way to add infinitely many vectors together, and we won't in this course.
