Steinitz Exchange Lemma Statement

The Steinitz exchange lemma states that in any vector space, if you have a linearly independent subset $S$ and a spanning set $T$, then:

  • $S$ is always smaller than $T$ or perhaps equal in size
  • By throwing away a number of elements of $T$ equal to the number of elements in $S$, you can put $S$ inside $T$ and still have a spanning set

In other words, you can put any linearly independent set inside a spanning set.

▶1:43

Previous Consequences

From the Steinitz exchange lemma, we saw two important consequences:

  1. If you have a finite dimensional vector space (i.e., a vector space with a finite spanning set), then not only does it have a finite basis, but all bases are finite and have the same size
  2. Every linearly independent set has size at most the dimension, and every spanning set has size at least the dimension. You get equality in either case if and only if you have a basis
▶2:21

Subspaces of Finite Dimensional Vector Spaces

If $V$ is a finite dimensional vector space and $U$ is a subspace of $V$, then:

  1. $U$ is also finite dimensional as a vector space
  2. The dimension of $U$ is always at most the dimension of $V$
  3. We have equality: $\dim U = \dim V$ if and only if $U = V$

So subspaces always have dimension that's less than or equal to the dimension of the finite dimensional vector space in which they sit, and if they're proper subspaces, it's strictly less.

▶3:26

Proof of Subspace Dimension

The proof works by constructing a finite basis for $U$ one element at a time, using consequences of the Steinitz exchange lemma to ensure the process terminates.

Step 1: If $U = \{0\}$, then we're done immediately.

Step 2: If not, pick some non-zero vector $u_1 \in U$. This single element set is certainly linearly independent.

Step k: After step $k$, assume we've constructed a linearly independent subset $\{u_1, u_2, \ldots, u_k\}$ of size $k$.

If this set is already a spanning set for $U$, then we're done because it's a basis of size $k$ and $k \leq \dim V$ (since a linearly independent set in $U$ is also linearly independent in $V$).

If not, move to the next step: find a new vector $u_{k+1}$ to add to the set, which is just something that lies outside the span of $\{u_1, \ldots, u_k\}$.

The set $\{u_1, \ldots, u_{k+1}\}$ must be linearly independent, otherwise we could write $u_{k+1}$ as a linear combination of the previous vectors, contradicting our choice.

This process has to terminate after at most $\dim V$ steps because at each stage we have a linearly independent set in $V$, and a linearly independent set can never grow bigger than the dimension.

Finally, if $\dim U = \dim V$, then $U$ has a basis of the same size as a basis of $V$, and by the previous corollary, the basis of $U$ is also a basis of $V$, so $U = V$.

Basis Extension

Another consequence of the Steinitz exchange lemma is that you can extend a basis from a subspace to a basis of the whole vector space.

If $V$ is finite dimensional and $U$ is a subspace, then for any basis $B_U$ of $U$, there exists a basis $B_V$ of $V$ which contains $B_U$.

This is called basis extension and will be referred to as such throughout the course.

▶11:55

Proof of Basis Extension

Let $B_U$ be the given basis of $U$ and let $B_V$ be any basis of $V$ (which exists since $V$ is finite dimensional).

Apply the second part of the Steinitz exchange lemma with $S = B_U$ (the linearly independent set) and $T = B_V$ (the spanning set).

This gives us a subset $T' \subseteq T$ with $|T'| = |T| - |S|$ such that $S \cup T'$ spans $V$.

Since $|T| = \dim V$ and $|S| = \dim U$, we have $|T'| = \dim V - \dim U$.

Let $B_V = B_U \cup T'$. We need to check that this is linearly independent.

Since it's a spanning set, by corollary 2, it will be linearly independent precisely when it has size at most $\dim V$.

Indeed, $|B_V| = |B_U| + |T'| = \dim U + (\dim V - \dim U) = \dim V$, so $B_V$ is a basis.

▶14:43

Basis Extension Strategy

It's always a good idea when you have a subspace sitting inside a vector space and need to do something with bases:

Instead of just taking an arbitrary basis for the vector space, always start with a basis for the subspace and extend it to a basis for the whole vector space. This is almost never a bad idea.

Steinitz Exchange Lemma

Let $V$ be an $F$-vector space, and let $S$ and $T$ be finite subsets of $V$ with $S$ linearly independent and $T$ spanning $V$. Then:

  1. $|T| \geq |S|$ (every spanning set is at least as big as every linearly independent set)
  2. There exists a subset $T' \subseteq T$ with $|T'| = |T| - |S|$ such that $S \cup T'$ spans $V$

This allows us to "exchange" vectors in $T$ that we don't like for vectors in $S$ that we do like, while preserving the spanning property.

▶15:43

Proof of Steinitz Exchange Lemma

The proof is an iterative procedure where we throw away one element from $T$ and put one element of $S$ in its place, doing this carefully to preserve spanning at every stage.

Let $S = \{u_1, u_2, \ldots, u_m\}$ and $T = \{v_1, v_2, \ldots, v_n\}$.

Base case: If $S$ is empty, we're done trivially.

Step 1: Put $u_1$ inside $T$. Since $T$ spans $V$, we can write $u_1 = \sum_{i=1}^n \lambda_i v_i$ with at least one $\lambda_i \neq 0$ (since $u_1 \neq 0$).

Without loss of generality, assume $\lambda_1 \neq 0$. Then we can rearrange to get $v_1 = \frac{1}{\lambda_1}u_1 - \sum_{i=2}^n \frac{\lambda_i}{\lambda_1}v_i$.

This tells us that $\{u_1, v_2, \ldots, v_n\}$ spans $V$ (since it contains $v_1$ in its span and certainly contains $v_2, \ldots, v_n$).

Step k+1: Suppose we've successfully put the first $k$ elements of $S$ inside $T$ and still have a spanning set. At step $k+1$, we want to put $u_{k+1}$ inside $T$.

Write $u_{k+1} = \sum_{j=1}^k \mu_j u_j + \sum_{i=k+1}^n \lambda_i v_i$.

Since $S$ is linearly independent, not all coefficients can be $\mu_j$ (otherwise we'd have a non-trivial linear combination of $S$ equal to zero). So some $\lambda_i$ must be non-zero.

Without loss of generality, assume $\lambda_{k+1} \neq 0$. Then $v_{k+1}$ can be made the subject, and $\{u_1, \ldots, u_{k+1}, v_{k+2}, \ldots, v_n\}$ spans $V$.

This process continues until all elements of $S$ are inside $T$. The only way it can terminate is if we run out of elements in $S$, so $m \leq n$.

At the final step, we have a spanning set $S \cup T'$ where $T'$ consists of the vectors from $T$ that we didn't throw away.

Linear Maps and Kernels

Now we turn to how these ideas relate to linear maps, which leads us to our next big theorem of the course.

Let $V$ be a finite dimensional $F$-vector space, $W$ be any $F$-vector space (not necessarily finite dimensional), and $\alpha: V \to W$ be a linear map.

Recall that we have:

  • Kernel of $\alpha$: $\ker(\alpha) = \{v \in V : \alpha(v) = 0\}$ (a subspace of $V$)
  • Image of $\alpha$: $\text{im}(\alpha) = \{\alpha(v) : v \in V\}$ (a subspace of $W$)
▶28:17

Rank-Nullity Theorem

Let $V$ be finite dimensional and $\alpha: V \to W$ be a linear map. Then:

  1. The image of $\alpha$ is finite dimensional
  2. The kernel of $\alpha$ is finite dimensional (since it's a subspace of the finite dimensional vector space $V$)
  3. These three dimensions satisfy: $\dim V = \dim(\ker(\alpha)) + \dim(\text{im}(\alpha))$
▶29:30

Nullity and Rank

Because these dimensions are so useful and we'll write them down many times, we give them special names:

  • Nullity of $\alpha$: $\text{null}(\alpha) = \dim(\ker(\alpha))$
  • Rank of $\alpha$: $\text{rank}(\alpha) = \dim(\text{im}(\alpha))$

So the rank-nullity theorem states: $\dim V = \text{rank}(\alpha) + \text{null}(\alpha)$

▶30:23

Proof of Rank-Nullity Theorem

The proof uses a consequence of the first isomorphism theorem, which states that $V/\ker(\alpha) \cong \text{im}(\alpha)$ as vector spaces.

It will be sufficient to prove the following lemma about dimensions of quotient spaces of finite dimensional vector spaces:

Lemma: If $V$ is finite dimensional and $U$ is a subspace of $V$, then $\dim(V/U) = \dim V - \dim U$.

If we apply this lemma with $U = \ker(\alpha)$, it says that $\dim(V/\ker(\alpha)) = \dim V - \dim(\ker(\alpha))$.

Since $V/\ker(\alpha) \cong \text{im}(\alpha)$, we have $\dim(\text{im}(\alpha)) = \dim V - \dim(\ker(\alpha))$, which gives us the rank-nullity theorem.

▶32:15

Proof of Quotient Space Dimension

This is a consequence of basis extension. Let $B_U = \{u_1, \ldots, u_m\}$ be a basis of $U$.

By basis extension, we can extend this to a basis $B_V = \{u_1, \ldots, u_m, v_{m+1}, \ldots, v_n\}$ of $V$.

Here $m = \dim U$ and $n = \dim V$.

We claim that $\{v_{m+1} + U, \ldots, v_n + U\}$ is a basis for the quotient space $V/U$.

Spanning: Every element of $V/U$ is represented by some vector $v \in V$. Write $v = \sum_{i=1}^m \mu_i u_i + \sum_{i=m+1}^n \lambda_i v_i$.

Then $v + U = \sum_{i=m+1}^n \lambda_i (v_i + U)$ since the first part lives in $U$ and is absorbed into the coset.

Linear Independence: Suppose $\sum_{i=m+1}^n \lambda_i (v_i + U) = 0$ in $V/U$.

This means $\sum_{i=m+1}^n \lambda_i v_i \in U$, so we can write it as $\sum_{j=1}^m \mu_j u_j$.

Rearranging gives $\sum_{i=m+1}^n \lambda_i v_i - \sum_{j=1}^m \mu_j u_j = 0$ in $V$.

Since $B_V$ is linearly independent, all coefficients must be zero, so $\lambda_i = 0$ for all $i$.

Size: The basis has size $n - m = \dim V - \dim U$ as required.

▶39:17

Alternative Proof Avoiding Quotient Spaces

If you just care about the rank-nullity theorem and aren't comfortable with quotient spaces, you can rearrange the ideas to avoid them altogether.

Direct proof sketch:

  1. Start with a basis $B_{\ker}$ for the kernel
  2. Extend to a basis $B_V$ for $V$
  3. Check that $\alpha$ applied to the new vectors (those not in $B_{\ker}$) forms a basis for the image

The process of checking this is very similar to the maneuvers in the quotient space proof.

▶40:45

Linear Pigeonhole Principle

A corollary of the rank-nullity theorem, which you might think of as a linear version of the pigeonhole principle:

If $V$ and $W$ are finite dimensional vector spaces of the same dimension $n$, and $\alpha: V \to W$ is a linear map, then the following conditions are equivalent:

  1. $\alpha$ is injective
  2. $\alpha$ is surjective
  3. $\alpha$ is an isomorphism

So for linear maps between spaces of the same dimension, either one of injectivity or surjectivity implies the other.

▶41:56

Proof of Linear Pigeonhole Principle

Recall that injectivity is equivalent to the kernel being trivial, i.e., $\text{null}(\alpha) = 0$.

Surjectivity is by definition the image being everything in $W$, i.e., $\text{rank}(\alpha) = n$.

By the rank-nullity theorem: $n = \text{rank}(\alpha) + \text{null}(\alpha)$.

If $\text{null}(\alpha) = 0$, then $\text{rank}(\alpha) = n$, so $\alpha$ is surjective.

If $\text{rank}(\alpha) = n$, then $\text{null}(\alpha) = 0$, so $\alpha$ is injective.

Therefore, either condition implies the other, and both together give us an isomorphism.

Extension of Functions from Basis

This proposition is independent of the Steinitz Exchange Lemma and is valid even for vector spaces that are not finite dimensional, but it requires that the vector space has a basis.

Let $V$ be a vector space (not necessarily finite dimensional) with a basis $B$. For any vector space $W$ and any function $f: B \to W$, there is a unique linear map $F: V \to W$ extending $f$.

In other words, a linear map is uniquely determined by what it does to a basis.

▶45:42

Proof of Function Extension

Existence: Since $B$ spans $V$, any vector $v \in V$ can be written as $v = \sum_{b \in B} \lambda_b b$ (a finite linear combination).

Define $F(v) = \sum_{b \in B} \lambda_b f(b)$.

This is well-defined because $B$ is a basis (in particular, linearly independent), so there is exactly one way to write each vector as a linear combination of $B$.

Linearity: Let $u, v \in V$ and $\lambda$ be a scalar. Write $u = \sum_{b \in B} \mu_b b$ and $v = \sum_{b \in B} \lambda_b b$.

Then $u + \lambda v = \sum_{b \in B} (\mu_b + \lambda \lambda_b) b$, so:

$$F(u + \lambda v) = \sum_{b \in B} (\mu_b + \lambda \lambda_b) f(b) = \sum_{b \in B} \mu_b f(b) + \lambda \sum_{b \in B} \lambda_b f(b) = F(u) + \lambda F(v)$$

Uniqueness: If $F'$ is another linear extension, then for any $v = \sum_{b \in B} \lambda_b b$:

$$F'(v) = F'\left(\sum_{b \in B} \lambda_b b\right) = \sum_{b \in B} \lambda_b F'(b) = \sum_{b \in B} \lambda_b f(b) = F(v)$$

So $F' = F$.

Coordinate Vector Isomorphism

For finite dimensional vector spaces, there is a special isomorphism that depends on the choice of basis:

If $V$ is an $F$-vector space of dimension $n$ and $B = \{v_1, v_2, \ldots, v_n\}$ is a basis, then there is an isomorphism $F_B: V \to F^n$ which sends the linear combination $\sum_{i=1}^n \lambda_i v_i$ to the vector $(\lambda_1, \lambda_2, \ldots, \lambda_n)$.

You might recognize this vector as the coordinate vector of $v$ in the basis $B$.
