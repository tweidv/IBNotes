▶03:23

Properties of Span

Several important properties of the span of a set $S$ in a vector space $V$:

  1. Span is always a subspace: For any subset $S$ of $V$, $\text{span}(S)$ is always a subspace of $V$.
  2. Smallest containing subspace: $\text{span}(S)$ is the smallest subspace that contains $S$ in the sense of inclusion.
  3. Transitivity property: If $S$ spans $V$ and $S \subseteq \text{span}(T)$, then $T$ also spans $V$.
  4. Empty set convention: By convention, $\text{span}(\emptyset) = \{0\}$ (the zero subspace).
  5. Union property: $\text{span}(S \cup T) = \text{span}(S) + \text{span}(T)$.
▶07:00

Span Examples in $\mathbb{R}^3$

Let $V = \mathbb{R}^3$ and consider two subsets:

  • $S = \{(1,0,0), (1,1,2)\}$
  • $T = \{(2,1,2), (0,1,2), (-1,2,4)\}$

Neither set spans the whole space, but they have the same span: the set of vectors of the form $(x,y,2y)$ where $x$ and $y$ are real numbers. This demonstrates that sets of different sizes can have the same span.

▶08:44

Span in Infinite Dimensional Space

Let $V$ be the space of all real sequences (functions from $\mathbb{N}$ to $\mathbb{R}$). Consider the subset $T = \{\delta_n : n \in \mathbb{N}\}$ where $\delta_n$ is the Kronecker delta function:

$$\delta_n(m) = \begin{cases} 1 & \text{if } n = m \\ 0 & \text{if } n \neq m \end{cases}$$

This set does not span $V$ because linear combinations must be finite sums, but general sequences have infinitely many non-zero terms. The span of $T$ consists of sequences that are eventually zero (zero for all but finitely many terms).

Linear Independence

A subset $S$ of a vector space $V$ is called linearly independent if for all linear combinations of $S$:

$$\sum_{s \in S} \lambda_s s = 0$$

where $\lambda_s \in F$ (the field) and only finitely many $\lambda_s$ are non-zero, we have that all coefficients $\lambda_s = 0$.

In other words, the only way to write the zero vector as a linear combination of $S$ is as the trivial linear combination where all coefficients are zero.

▶14:35

Properties of Linear Independence

  1. Zero vector property: If the zero vector is in $S$, then $S$ is always linearly dependent.
  2. Finite sets: For finite sets $S = \{v_1, v_2, \ldots, v_n\}$, linear independence means that if $\sum_{i=1}^n \lambda_i v_i = 0$, then all $\lambda_i = 0$.
  3. Infinite sets: An infinite set $S$ is linearly independent if and only if every finite subset of $S$ is linearly independent.
  4. Subset property: Every subset of a linearly independent set is itself linearly independent.
▶18:33

Linear Independence Examples

Example 1: In $\mathbb{R}^3$, the set $S = \{(1,0,0), (1,1,2)\}$ is linearly independent, while $T = \{(2,1,2), (0,1,2), (-1,2,4)\}$ is linearly dependent because:

$$1 \cdot (2,1,2) - 5 \cdot (0,1,2) + 2 \cdot (-1,2,4) = (0,0,0)$$

Example 2: The set $\{\delta_n : n \in \mathbb{N}\}$ of Kronecker delta functions in the space of real sequences is linearly independent.

Basis

A subset $S$ of a vector space $V$ is a basis if $S$ is both linearly independent and a spanning set for $V$.

Bases represent the confluence of linear independence and spanning properties, leading to many important results in linear algebra.

▶23:32

Basis Examples

Standard basis for $F^n$: The set $\{e_1, e_2, \ldots, e_n\}$ where $e_i$ has 1 in the $i$-th position and 0 elsewhere.

Alternative basis for $F^3$: $\{(1,0,0), (1,1,0), (1,1,1)\}$ is also a basis for $F^3$.

Polynomial basis: For the space of real polynomials $P(\mathbb{R})$, the set $\{p_n : n \geq 0\}$ where $p_n(x) = x^n$ is a basis.

Finite Spanning Set Contains a Basis

If $S$ is a finite spanning set for a vector space $V$, then there exists a subset $S' \subseteq S$ such that $S'$ is a basis for $V$.

This shows that having a finite spanning set is equivalent to having a finite basis, justifying our definition of finite dimensionality.

▶28:32

Proof of Spanning Set Proposition

If $S$ is already linearly independent, we're done. Otherwise, there exists a non-trivial linear combination equal to zero:

$$\sum_{i=1}^n \lambda_i v_i = 0$$

where at least one $\lambda_i \neq 0$. Without loss of generality, assume $\lambda_n \neq 0$. Then we can solve for $v_n$:

$$v_n = -\frac{1}{\lambda_n} \sum_{i=1}^{n-1} \lambda_i v_i$$

This shows that $v_n$ is in the span of the other vectors, so we can remove $v_n$ from $S$ and still have a spanning set. We repeat this process until we obtain a linearly independent set, which must happen in finitely many steps since $S$ is finite.

▶33:14

Steinitz Exchange Lemma

Let $V$ be a finite-dimensional vector space over a field $F$, and let $S$ and $T$ be finite subsets of $V$ with $S$ linearly independent and $T$ spanning $V$. Then:

  1. $|T| \geq |S|$ (every spanning set is at least as big as every linearly independent set)
  2. There exists a subset $T' \subseteq T$ with $|T'| = |T| - |S|$ such that $S \cup T'$ spans $V$

All Bases Have the Same Size

In a finite-dimensional vector space, every basis is finite and all finite bases have the same size.

This follows from applying the Steinitz Exchange Lemma with two different bases, showing that each has size at most the other.

▶42:05

Dimension

The dimension of a finite-dimensional vector space $V$, denoted $\dim V$, is the size of any basis of $V$.

This is well-defined because all bases have the same size.

▶43:24

Dimension Examples

$\dim F^n = n$: The standard basis has $n$ elements.

Complex numbers: $\mathbb{C}$ has dimension 1 as a complex vector space (basis: $\{1\}$) but dimension 2 as a real vector space (basis: $\{1, i\}$).

▶44:54

Size Bounds for Subsets

Let $V$ be a finite-dimensional vector space with dimension $n$. For finite subsets $S$ and $T$ of $V$:

  1. If $S$ is linearly independent, then $|S| \leq n$
  2. If $T$ spans $V$, then $|T| \geq n$

Equality holds if and only if $S$ or $T$ respectively is a basis.

▶47:32

Proof of Size Bounds

The inequalities follow immediately from the Steinitz Exchange Lemma applied with a basis for $V$.

For the equality case: If $|S| = n$ and $S$ is linearly independent, then by the exchange lemma, we can replace all vectors in any basis with the vectors in $S$, showing that $S$ spans $V$ and hence is a basis. The proof for spanning sets is similar.

▶50:30

Infinite Dimensional Vector Spaces

Note: This course will not prove that every vector space has a basis. While some infinite-dimensional vector spaces do have explicit bases (like the polynomial space), the general existence of bases for arbitrary vector spaces depends on the Axiom of Choice and is beyond the scope of this course.

When working on problems, only assume a vector space has a basis if you can explicitly write one down or if the problem states the space is finite-dimensional.
