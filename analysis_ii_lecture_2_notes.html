Uniform Convergence (Recall)

Let $(f_n)$ be a sequence of functions defined on a set $E$, and let $f$ be a function on $E$. We say that $f_n$ tends to $f$ uniformly on $E$ if:

$$\sup_{x \in E} |f_n(x) - f(x)| \to 0 \quad \text{as } n \to \infty$$

This means the supremum of the difference over all $x \in E$ tends to zero. This is equivalent to saying: for every $\epsilon > 0$, there exists $N$ (depending only on $\epsilon$) such that for all $n \ge N$ and all $x \in E$:

$$|f_n(x) - f(x)| < \epsilon$$

▶02:35

Uniform Convergence Implies Pointwise Convergence

If $f_n \to f$ uniformly on $E$, then $f_n \to f$ pointwise on $E$.

Proof: If we fix a point $x \in E$, then for any $\epsilon > 0$, uniform convergence gives us an $N$ such that for all $n \ge N$:

$$|f_n(x) - f(x)| \le \sup_{y \in E} |f_n(y) - f(y)| < \epsilon$$

This shows pointwise convergence at $x$. Since $x$ was arbitrary, pointwise convergence holds everywhere on $E$.

▶04:29

Strategy for Finding Uniform Limits

This remark gives us a useful strategy for computing or checking uniform limits for explicitly defined sequences:

  1. First, check if a pointwise limit exists. Compute $\lim_{n \to \infty} f_n(x)$ for each fixed $x$. If no pointwise limit exists, then there is no uniform limit either.
  2. If a pointwise limit $f$ exists, it is the only possible candidate for the uniform limit (since limits are unique).
  3. Then check the supremum condition: Verify whether $\sup_{x \in E} |f_n(x) - f(x)| \to 0$.

Uniformly Cauchy Sequence

Let $(f_n)$ be a sequence of functions defined on a set $E$. We say the sequence is uniformly Cauchy on $E$ if:

For every $\epsilon > 0$, there exists $N \in \mathbb{N}$ (depending on $\epsilon$) such that for all $n, m \ge N$:

$$\sup_{x \in E} |f_n(x) - f_m(x)| < \epsilon$$

This is analogous to the Cauchy criterion for sequences of real numbers, but now we require the condition to hold uniformly over all $x \in E$ (hence the supremum).

Cauchy Criterion for Uniform Convergence

Let $(f_n)$ be a sequence of functions $f_n : E \to \mathbb{R}$ defined on a common domain $E$. Then the following are equivalent:

  1. $(f_n)$ converges uniformly on $E$ (to some function $f$)
  2. $(f_n)$ is uniformly Cauchy on $E$

In other words: A sequence of functions converges uniformly if and only if it is uniformly Cauchy.

This is extremely useful because it allows us to check convergence without knowing the limit function in advance.

Cauchy Criterion (Forward Direction)

To prove: If $f_n \to f$ uniformly on $E$, then $(f_n)$ is uniformly Cauchy.

Proof: Suppose $f_n \to f$ uniformly. Let $\epsilon > 0$ be given. By uniform convergence, there exists $N$ such that for all $n \ge N$:

$$\sup_{x \in E} |f_n(x) - f(x)| < \epsilon$$

Now for any $n, m \ge N$ and any $x \in E$, by the triangle inequality:

$$|f_n(x) - f_m(x)| \le |f_n(x) - f(x)| + |f_m(x) - f(x)|$$

Since $x$ was arbitrary, we can take the supremum over $x \in E$:

$$\sup_{x \in E} |f_n(x) - f_m(x)| \le \sup_{x \in E} |f_n(x) - f(x)| + \sup_{x \in E} |f_m(x) - f(x)| < \epsilon + \epsilon = 2\epsilon$$

Since $\epsilon$ was arbitrary (we could have started with $\epsilon/2$), this shows $(f_n)$ is uniformly Cauchy. $\square$

Cauchy Criterion (Reverse Direction)

To prove: If $(f_n)$ is uniformly Cauchy on $E$, then $(f_n)$ converges uniformly on $E$.

Proof: Suppose $(f_n)$ is uniformly Cauchy.

Step 1: Find a pointwise limit. For each fixed $x \in E$, the sequence $(f_n(x))$ is a Cauchy sequence of real numbers (since uniform Cauchy implies pointwise Cauchy). By completeness of $\mathbb{R}$, this sequence converges to some limit. Define:

$$f(x) = \lim_{n \to \infty} f_n(x)$$

This gives us a function $f : E \to \mathbb{R}$, and we have $f_n \to f$ pointwise on $E$.

Step 2: Show uniform convergence. Let $\epsilon > 0$ be given. Since $(f_n)$ is uniformly Cauchy, there exists $N$ such that for all $n, m \ge N$:

$$\sup_{x \in E} |f_n(x) - f_m(x)| < \epsilon$$

This means that for every $x \in E$: $|f_n(x) - f_m(x)| < \epsilon$ for all $n, m \ge N$.

Now fix $n \ge N$ and let $m \to \infty$. By pointwise convergence, $f_m(x) \to f(x)$, so:

$$|f_n(x) - f(x)| = \lim_{m \to \infty} |f_n(x) - f_m(x)| \le \epsilon$$

This holds for all $x \in E$, so:

$$\sup_{x \in E} |f_n(x) - f(x)| \le \epsilon$$

Since $\epsilon$ was arbitrary and this holds for all $n \ge N$, we have shown $f_n \to f$ uniformly. $\square$

$f_n(x) = \frac{\sin(nx)}{n}$ on $[0,1]$

Consider the sequence $f_n(x) = \frac{\sin(nx)}{n}$ defined on the interval $[0,1]$.

Pointwise limit: For any fixed $x \in [0,1]$, since $|\sin(nx)| \le 1$:

$$|f_n(x)| = \left|\frac{\sin(nx)}{n}\right| \le \frac{1}{n} \to 0$$

So $f_n \to 0$ pointwise on $[0,1]$.

Uniform convergence: Now check the supremum:

$$\sup_{x \in [0,1]} |f_n(x) - 0| = \sup_{x \in [0,1]} \frac{|\sin(nx)|}{n} \le \frac{1}{n} = \frac{1}{n} \to 0$$

In fact, the supremum equals $n$ (the maximum value of $|\sin(nx)|$ is 1), so:

$$\sup_{x \in [0,1]} |f_n(x)| = n \not\to 0$$

Conclusion: The sequence converges pointwise to 0 but does not converge uniformly on $[0,1]$.

▶29:32

$f_n(x) = \frac{x}{n}$ on $\mathbb{R}$

Consider $f_n(x) = \frac{x}{n}$ defined on all of $\mathbb{R}$.

Pointwise limit: For any fixed $x \in \mathbb{R}$:

$$\lim_{n \to \infty} \frac{x}{n} = 0$$

So $f_n \to 0$ pointwise on $\mathbb{R}$.

Uniform convergence: Check the supremum:

$$\sup_{x \in \mathbb{R}} \left|\frac{x}{n} - 0\right| = \sup_{x \in \mathbb{R}} \frac{|x|}{n} = +\infty$$

Since the domain is unbounded, for any fixed $n$, we can make $|x|/n$ arbitrarily large by choosing $x$ large enough.

Conclusion: The sequence converges pointwise to 0 but does not converge uniformly on $\mathbb{R}$.

▶31:07

$f_n(x) = \frac{x}{n}$ on $[-a, a]$

Now consider the same sequence $f_n(x) = \frac{x}{n}$ but restricted to a bounded interval $[-a, a]$ for some finite $a > 0$.

Pointwise limit: As before, $f_n \to 0$ pointwise (restriction doesn't change pointwise limits).

Uniform convergence: Now the supremum is:

$$\sup_{x \in [-a,a]} \left|\frac{x}{n}\right| = \frac{a}{n} \to 0$$

Conclusion: The sequence does converge uniformly to 0 on any bounded interval $[-a, a]$.

This example illustrates that the domain matters! The same sequence can converge uniformly on one domain but not on another.

Continuity is Preserved Under Uniform Convergence

Let $(f_n)$ and $f$ be functions defined on an interval $[a,b]$. Suppose:

  1. $f_n \to f$ uniformly on $[a,b]$
  2. Each $f_n$ is continuous at a point $x_0 \in [a,b]$

Then $f$ is continuous at $x_0$.

Note: We only need continuity at a single common point $x_0$ for all the $f_n$. If all $f_n$ are continuous everywhere, then $f$ is continuous everywhere.

▶37:15

Continuity is Preserved Under Uniform Convergence

Proof: Let $\epsilon > 0$ be given. We need to show that $f$ is continuous at $x_0$.

Step 1: By uniform convergence of $f_n \to f$, there exists $N$ such that for all $n \ge N$:

$$\sup_{y \in [a,b]} |f_n(y) - f(y)| < \epsilon$$

In particular, this holds for $n = N$.

Step 2: Since $f_N$ is continuous at $x_0$, there exists $\delta > 0$ such that for all $y \in [a,b]$ with $|y - x_0| < \delta$:

$$|f_N(y) - f_N(x_0)| < \epsilon$$

Step 3: Now for any $y$ with $|y - x_0| < \delta$, we use the triangle inequality:

$$|f(y) - f(x_0)| \le |f(y) - f_N(y)| + |f_N(y) - f_N(x_0)| + |f_N(x_0) - f(x_0)|$$

By Step 1, the first and third terms are each less than $\epsilon$ (since $n = N \ge N$). By Step 2, the middle term is less than $\epsilon$. Therefore:

$$|f(y) - f(x_0)| < \epsilon + \epsilon + \epsilon = 3\epsilon$$

Since $\epsilon$ was arbitrary, this shows $f$ is continuous at $x_0$. $\square$

Where would this fail for pointwise convergence? The issue is that with only pointwise convergence, the $N$ in Step 1 would depend on $y$, and we couldn't use the same $N$ for all $y$ near $x_0$.

Uniform Limits of Continuous Functions are Continuous

Let $(f_n)$ and $f$ be functions defined on an interval $[a,b]$. Suppose:

  1. $f_n \to f$ uniformly on $[a,b]$
  2. Each $f_n$ is continuous on $[a,b]$ (for every $n$)

Then $f$ is continuous on $[a,b]$.

This is a direct consequence of the previous theorem applied at every point in $[a,b]$.

Significance: We've found a notion of convergence (uniform convergence) that preserves the property of continuity! This is what we set out to achieve. The space of continuous functions is "closed" under taking uniform limits.

Completeness of $C[a,b]$

Let $C[a,b]$ denote the space of all continuous real-valued functions on the interval $[a,b]$. This is a vector space (we can add continuous functions and multiply by scalars).

If $(f_n)$ is a sequence in $C[a,b]$ that is uniformly Cauchy, then $(f_n)$ converges uniformly to some function $f \in C[a,b]$.

In other words: $C[a,b]$ is complete with respect to uniform convergence.

Proof: Combine Theorem 1.1 (Cauchy criterion) and Corollary 1.1 (continuity preservation):

  • By the Cauchy criterion, if $(f_n)$ is uniformly Cauchy, then $f_n \to f$ uniformly for some function $f$.
  • By Corollary 1.1, since each $f_n$ is continuous and convergence is uniform, the limit $f$ is also continuous.
  • Therefore $f \in C[a,b]$, and the sequence converges to a function within the space.

$\square$

Completeness Analogy

Compare this result to the completeness of $\mathbb{R}$:

  • For real numbers: Every Cauchy sequence of real numbers converges to a real number. There are "no holes" in $\mathbb{R}$.
  • For $C[a,b]$: Every uniformly Cauchy sequence of continuous functions converges uniformly to a continuous function. There are "no holes" in $C[a,b]$ with respect to uniform convergence.

This is remarkable because $C[a,b]$ is an infinite-dimensional vector space, yet it has this completeness property.

If we removed a single point from $\mathbb{R}$ (e.g., $\mathbb{R} \setminus \{0\}$), it would no longer be complete—there would be Cauchy sequences (like $1/n$) that want to converge to the missing point. Similarly, if we removed a function from $C[a,b]$, we could have uniformly Cauchy sequences trying to converge to it.
